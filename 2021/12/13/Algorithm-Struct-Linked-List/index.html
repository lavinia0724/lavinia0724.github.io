Linked List

malloc

  • memory allocation
  • 配置指定大小的記憶體空間
    • 配置的空間處於尚未初始化的狀態
  • 動態的宣告記憶體空間
    • 要多少開多少

free

  • 釋放之前使用 malloc 所配置的記憶體空間
  • memory leak
    • 程式未釋放已不再使用的內部記憶體空間
    • 用完記憶體後沒有 free 掉
  • 在程式執行完後會自動把記憶體還回去

Doubly Linked List

  • 雙向 Linked List
  • 左指標與右指標
    • 右指標接下個節點
    • 左指標接上個節點

Insertion

  • 插入
    • 遍歷找到要插入新節點的位址
    • 前一節點的右指標 -> 新節點
    • 新節點的左指標 -> 前一節點
    • 新節點的右指標 -> 下一節點
    • 下一節點的左指標 -> 新節點

Deleting

  • 刪除
    • 遍歷找到要刪除節點的位址
    • 前一節點的右指標 -> 被刪除節點之下一節點
    • 下一節點的左指標 -> 前一節點
    • free 歸還被刪除之節點的記憶體空間
