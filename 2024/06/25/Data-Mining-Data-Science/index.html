Primary Sources of Big Data

  • 個人社交媒體貼文、評論、影片…
  • 公司業務流程、商業交易數據
  • sensors，例如 IoT、image/video data

大多 Big Data 都與 Cloud 有關，或可說是基於 Cloud 的，因此

The Era of Big Data

  • 導致機器學習爆炸式增長
  • 大數據是高容量，高速度的訊息資產，需要新的處理方式來增強決策，洞察和流程優化

由於迅速累積的大量資料，長遠的影響了 Deep Learning (Artificial Intelligence) 的誕生，因此

新 AI 的崛起

Now, with

  • 從大數據中持續發現知識
  • 提高決策質量，優化流程，獲得總體洞察能力

Illustration for Mining Capabilities

  • Association
    – Apriori, FP-Tree
  • Classification
    – Decision tree, SVM, Neural Network, KNN, Bayes
  • Clustering
    – K-means, PAM, DBScan
  • Anomaly detection
    – A combination of analysis methods

Procedure of Data Mining

  1. 獲取並查看數據
  2. 決定目標 (一個延伸目標和一個可達成的目標)
  3. Data cleaning
  4. 選擇 Data granularity、Data 特徵
  5. 應用 Data mining method (classification, clustering…)
  6. 決定輸出內容和形式
  7. 解釋結果，內容改進

Handling Data

  • Data source
    • how and in what form
    • 合法性、可信度
    • 質量
    • 是否足以實現數據分析目標
    • 特徵/屬性選擇
    • 了解和處理數據在數據分析中非常重要

Start with a table of data

collection of data objects and their attributes

  • attribute 是一個對象的屬性或特徵
  • attribute 也被稱為 column、variable、field、characteristic、feature..
  • a collection of attributes 可以描述一個 object
  • object 也被稱為 row、tuple、record、point、case、sample、entity、instance..
Data granularity
  • Distinction between attributes (ID) and attribute values (A12…)
  • Same attribute can be mapped to different values
Discrete Attribute
  • 離散屬性
  • 只有有限或可數的無限 set of values
  • 通常用整數變量表示
Continuous Attribute
  • 連續屬性
  • 屬性值為實數
  • 連續數性通常用浮點變量來表示

Types of data sets

  • Record
    • Data Matrix
    • Document Data
    • Transaction Data
  • Graph
    • World Wide Web
    • Molecular Structures
  • Ordered
    • Spatial Data
    • Temporal Data
    • Sequential Data
    • Genetic Sequence Data
