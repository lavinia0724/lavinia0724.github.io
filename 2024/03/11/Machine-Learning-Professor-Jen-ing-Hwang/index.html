Regression 必回傳數值

Linear Regression


圖片來源

回歸分析是在找 y = β0 + β1x 中的 β0 與 β1
讓目標函數與損失函數 loss function 找到的模型最種 error 越小越好
找法為：

帶入資料 (xi, yi), i = 1, …, n，會產生
ŷi = β0 + β1xi，i = 1, …, n

ŷi: 為預估出的值，和真實資料會有誤差 (error)，因此可得下式:

εi = yi — ŷi

其中一種常見方法為最小平方法，希望誤差的平方越小越好，稱為 S

Attribute

屬性，指標的的特徵
在 input 中稱為 label
在 output 中稱為 target

attribute = feature = variable = characteristic = field

有兩種型別：

  1. categorical 分類
  2. numeric 數值

Linear model

w1x1 + w2x2 + b = 0

Hypothesis: a set of linear func
Learning Algorithm: PLA

Rule-Based model

Hypothesis: a set of rules
Learning Algorithm: Decision Tree

Instance-Based model

基於案例

Hypothesis: an implicit hypothesis set
Learning Algorithm: k-nearest neighbor（kNN），找距離目標最近的 k 個物件

Decision Tree 決策樹

Bayesian Learning

基於貝式理論，條件機率，根據前提猜測結果

Clustering

不知道要分幾群，和分類不同，沒有標準答案，用很多種演算法實作
