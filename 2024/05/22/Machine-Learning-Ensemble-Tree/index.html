Ensemble Tree(multiple decision tree)

Real-Valued attribute 可以預先被離散的分為大、中、小範圍
也可以將數據分為 A < c 和 A >= c 的範圍，將 Real-Value 分為兩個區間
這些分割的 information gain 與分類特徵類似，不同區間的 information gain 會互相進行比較，從而訓練

Example

對收入值進行排序，並考慮每兩個相鄰點之間的每個中點，例如介於 120k 之間

計算每個介於 120k 之間的中點 information gain

Threshold split selection algorithm

  • Step 1: Sort the values of a real valued feature
    • Let 表示排序值
  • Step 2:
    • For i = 1 … N-1
    • 分割
    • 計算每個分割的 information gain
  • 選擇 information gain 最高的

Information Gain of a Real-Valued Input

和 ID3 算法一樣，只是 Information Gain 中的 Entropy 替換成 Variance

Regression Problem(Real-Value Output)

Bagging

Bagging 是 Bootstrap Aggregating 的縮寫
Bagging 由兩部分組成，bootstrapping(引導) and aggregating(聚合)

Bootstrapping:
一種隨機抽樣的替換方法

Aggregating:
將不同子集上訓練的多個版本的模型結合起來，進行最終預測

Bagging 重點是取樣的不同
