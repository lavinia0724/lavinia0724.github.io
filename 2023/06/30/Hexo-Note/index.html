環境配置

  1. 下載 node.js
  2. 安裝 Hexo CLI
  3. 建立 Project
  4. 安裝 Hexo 相關套件
  5. 啟動 Hexo

    此時 terminal 會出現 Hexo is running at http://localhost:4000/
    若連線 http://localhost:4000/ 出現了 Hexo 預設畫面，則代表成功

Hexo 主題修改

Hexo 官方主題模板

  1. 選擇喜歡的主題後下載解壓縮於 <projectname>\themes
  2. 開啟位於 <projectname>\_config.yml 檔案，找到 theme 並修改
  3. 重啟 Hexo server，此時網頁更新為下載的模板樣
  4. 針對位於 <projectname>\themes\redefine\_config.yml 進行修改，就可以將模板個人化成自己的部落格ㄌ

佈署到 Github

  1. 安裝 Hexo deployer git 套件
  2. 修改位於 <projectname>\_config.yml 的
  3. 利用 Hexo CLI 將本地更新好的整體網頁佈署到 Github

新增文章

  • layout 為模板種類，預設有以下三種
    1. draft.md 新增草稿
    2. page.md 新增頁面
    3. post.md 新增文章 (預設若無寫 [layout] 時為此方式)
  • 更多新增文章方式：Hexo 新增文章和頁面
