DropDownList

定義這個下拉式選單做什麼

接著將這個下拉式選單實作出來

dataTextField: “text”：為讀取顯示方才定義的 “text”
dataValueField: “value”：透過方才定義的 “value” 決定實際放置位置
dataSource: data：讀取資料來源

因為使用 Kendo 套件就需要用官方指定的方式操作，所以要進入 Kendo DropDownList API 來得知如何取值和設定值
