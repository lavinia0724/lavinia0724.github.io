ASP.NET MVC Framework - View

MVC：Model-View-Controller 軟體設計模式

其中包含以下三大面向：

  1. Controller：從 Model 拿資料再將其提供給 View

  2. View用：呈現使者看到的畫面

  3. Model：處理商業邏輯，提供資料
