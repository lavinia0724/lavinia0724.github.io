解題觀念

Greedy - 區間最小覆蓋

可以參考我的課程簡報：Greedy & Graph

題目敘述

給定多組線段座標 [Li, Ri] (在 x 軸上)
請選擇以最少數量的線段,使得選擇的線段完全覆蓋 [0, M]

Input

輸入的第一行有一整數 t,表示測茲的數量
接下來每組測資的第一行為空白行
第二行為一整數 M
表示需覆蓋的線段最右值 (1 <= M <= 5000)
接下來有多行
每行有兩個整數 Li、Ri
表示可用線段的最左值及最右值
(|Li|、|Ri| <= 50000,i <= 100000)
若 Li == Ri == 0 則此組測資結束

Output

每組測資輸出
第一行輸出最少須選擇多少線段以覆蓋 [0, M]
接下來多行輸出選擇的線段座標
(依據 Li 由小到大排序)
如果所有可用線段無法使 [0, M] 被完全覆蓋則輸出 “0”

每組輸出之間需相隔一行
