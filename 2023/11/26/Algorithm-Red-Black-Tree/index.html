Red-Black Tree

規則

  1. 根節點一定是黑色
  2. 葉節點一定是黑色的 NULL
  3. 紅節點的兩個兒子一定是黑色，不能有兩個紅節點相連
  4. 從任何節點出發，其往下到葉節點的所有路徑，黑色節點數目要相同

Insertion

  • 插入的節點必為紅色
  • 規則：
    1. 初始紅黑樹為空，插入紅節點成為根節點，將節點改為黑色

      圖片來源
    2. 在黑節點上與紅節點相接，不需做其他調整

      圖片來源
    3. 接在紅節點上與紅節點相接，若叔叔為紅色，塗黑父節點，將祖父改為紅色，叔叔改為黑色，再繼續根據規則維護

      圖片來源
    4. 接在紅節點上與紅節點相接，若叔叔為黑色，自己在紅父節點右邊，先以父節點左旋，再繼續根據規則維護

      圖片來源
    5. 接在紅節點上與紅節點相接，若叔叔為黑色，自己在紅父節點左邊，先將父節點改為黑色，再將祖父改成紅色，以祖父為軸進行右旋

      圖片來源

Deletion

  • 規則：
    1. 刪除紅節點，不違反任何規則，無須進行維護

      圖片來源
    2. 刪除黑節點，遞補上來的節點為紅，將紅節點塗黑（根的狀況）

      圖片來源
    • 刪除黑節點，若遞補上來的節點為黑，須補足路徑上缺失的黑色：
    1. 若兄弟為紅，將兄弟塗黑，父親塗紅，再以父節點為軸左旋，再繼續根據規則維護

      圖片來源
    2. 若兄弟為黑，兄弟的兩個子節點都為黑，將兄弟塗紅，此時若父親為紅，將父親塗黑，但若父親為黑，需再繼續根據規則維護

      圖片來源
    3. 若兄弟為黑，且兄弟的左兒子為紅，右兒子為黑，將兄弟塗紅，左兒塗黑，並以兄弟為軸右旋，再繼續根據規則維護

      圖片來源
    4. 若兄弟為黑，且兄弟的右兒子為紅，將兄弟塗成與父親相同的顏色，父親塗黑，兄弟的右兒子塗黑，以父親為軸左旋

      圖片來源
