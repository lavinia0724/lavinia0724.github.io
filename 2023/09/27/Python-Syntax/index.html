Python Introduction

Introduction

直譯式語言

  • 程式一行一行執行，不須像編譯語言先由編譯器編譯成機器碼
  • 開發速度快、靈活度高
  • 執行速度慢


圖片來源

動態語言 Dynamic Language

  • 變數不須指定型態（int、string、double…）
  • 執行時才確定變數型別
  • 執行過程中變數型態可以改變
    • ex：int -> string

強型別語言

  • 不容忍隱性的型別轉換
  • 比較嚴謹，方便 debug
  • 強型別和弱型別不是二元論，有程度差異

排版

  • 縮排很重要！
  • Python 不用 Block { }
  • 排版在 Python 中非常重要
  • 使用 tab，而非空白
    • 不能將 tab 與空白混用

Installation

  1. https://www.python.org/downloads/

  2. 推薦選擇加入 python.exe 到 PATH 裡，方便系統知道要到哪條路徑底下找執行檔

  3. 檢查是否下載成功，可以打開 cmd 輸入
    會顯示你所下載的 Python 當前版本
