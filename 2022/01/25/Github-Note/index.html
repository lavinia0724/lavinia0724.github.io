Instruction

  1. 新建本地數據庫

  2. 將檔案加入索引中

  3. 查看檔案
    我都用來回憶一下剛剛推了什麼，方便寫 commit

  4. 將索引中的檔案提交到本地數據庫
    commit 可以寫一些更新的內容，方便知道本次更新了什麼，像是註解，方便維護

  5. 新增分支

  6. 新增遠端連線

  7. 查看遠端連線

  8. 把本地數據庫檔案推上 github

  9. 把 github 上的檔案拉到本地數據庫

引用外部檔案

如果推上 github 的檔案中有 clone 別人的檔案，在推上 github 後會顯上白色箭頭且無法打開
原因是在本機的資料夾中有 .git 隱藏資料夾

解決辦法：

  1. 刪除資料夾中的 .git 隱藏資料夾

在有 .git 的資料夾路徑下，點擊上方的 三個點，然後選擇 檢視， 點選 顯示隱藏的檔案、資料夾及磁碟機，
按確定後就會出現隱藏的資料夾了

  1. 在此資料夾的父資料夾路徑下，執行以下指令

刪除完 cache 後，就可以普通的回到推本機資料上 github 的指令環節了
