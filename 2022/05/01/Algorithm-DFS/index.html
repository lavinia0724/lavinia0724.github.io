Depth-First Search DFS 深度優先搜尋

  • 不斷找出尚未遍歷的點當作起點
    • 把起點塞入 stack
    • 重複以下步驟，直到 stack 裡沒有東西為止：
      1. 從 stack 中取出一點
      2. 找出與此點相鄰的點、且尚未遍歷的點，並將這些點依序塞入 stack
